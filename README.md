# H1 Это проектная работа, спринт 03.

Данное веб-приложение содержит информацию для людей о том, как сосредоточиться.

## H2 Основная область интерфейса:
- Списки текста с какой-то информацией на тему сайта.

## H2 Шапка:
- Надпись для привлечения внимания

## H2 Подвал:
- Надпись для привлечения внимания

## H2 декоративная рамка видеосъёмки и надпись REC:
- Надпись скрывается в светлой теме
- Декоративная рамка реализована в виде самостоятельных тэгов, игнорируя класс decorated-zone. Это сделано НАМЕРЕННО, т.к. данный подход более гибкий, а "лишние" тэги в разметке всё равно, и так, и так, будут (тэг "REC"). С таким подходом через задание рамки отдельными тэгами, можем сделать и все 4 уголка рамки, если захотим.

## H2 Меню переключения тем оформления в верхней части интерфейса:
- Позволяет переключать темы оформления сайта между светлой, авто, и тёмной.
- Учитывает автоматизацию переключения тем в соответствии с настройками предпочитаемой темы оформления в браузере (prefers-color-scheme), либо в операционной системе .

Данное приложение реализовано с поддержкой адаптивного отображения интерфейса
для смартфонов, планшетов, и более крупного размера
экрана.
